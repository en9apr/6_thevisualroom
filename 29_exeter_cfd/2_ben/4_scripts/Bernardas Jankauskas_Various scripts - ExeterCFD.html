Scripts for various tasks

Here I will put various bash scripts that I write to help me with my work. Will aim to have them OpenFOAM specific. Note that scripts will usually be one task specific, but might have useful ideas for your own cases.

Simple script to run multiple cases

In this script I aimed to run multiple cases. At each run a parameter $TaInitial is adjusted inside one of the dictionary files and loop is run until the final parameter value of $TaFinal is reached. After each simulation a whole case directory is copied to another folder to store all of the data.

Usage: ./Allrun A TaInitial TaFinal
