1.2.1. Associativity¶

  • Right-to-left means the value on the right is evaluated first
  • Left-to-right means the value on the left is evaluated first

1.2.2. Order of Precedence¶

The order of precedence for operators:

  • Bracket
  • Arithmetic
  • Relational

  • Assignment
