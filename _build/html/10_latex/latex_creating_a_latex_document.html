3.1.1. Basic Document Setup¶

The document is setup like this:

Code Meaning 
\documentclass[11pt]{article}
The preamble 
\begin{document}
This is the body of the document
\end{document}
Body of the document 
Inline maths $(x+1)$
Inline maths 
Display maths $$(x+1)$$
Display (centred on it’s own line) maths 
Line 1. \\
Line 2.
New line

3.1.2. Files¶

There are five files created, the two most important ones are in bold (.pdf and .tex)

  • filename.aux
  • filename.txt
  • filename.pdf (output)
  • filename.synctex.gz
  • filename.tex (latex command file)
