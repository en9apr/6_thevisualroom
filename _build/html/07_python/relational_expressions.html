3.5.1. Relational Operators¶

The values of two arithmetic expressions can be compared using the operators:

  • ==
  • !=
  • <
  • >
  • <=
  • >=

These return either:

  • True
  • False

3.5.2. Order of Precedence¶

Now we have looked at all the operators, we can give the order of precedence:

Order Name Symbol
1 Brackets ()
2 Unary not - +
3 Binary Arithmetic 1 **
4 Binary Arithmetic 2 * / %
5 Binary Arithmetic 3 + -
6 Relational 1 < <= => >
7 Relational 2 == !=
8 Logical And and
9 Logical Or or
10 Assignment = += -= *= /= %= **=

Basically:

  • Brackets
  • Unary
  • Arithmetic 1,2,3
  • Relational 1,2
  • Logical 1,2
  • Assignment
